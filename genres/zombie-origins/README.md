# Zombie Origins Template

## Overview
A template for creating zombie games with origin storylines, character development, and narrative-driven survival mechanics.

## Core Requirements
1. Zombie origin narratives with branching storylines
2. Character progression with survival skills and abilities
3. Resource management in post-apocalyptic settings
4. Horde encounters with varied zombie types

## Essential Systems

### Origin Story System
- Purpose: Deliver compelling zombie outbreak narratives
- Components: Story manager, character arcs, plot branches
- Implementation: Event-driven narrative with multiple possible origins

### Survival Progression
- Purpose: Enable character growth through survival challenges
- Components: Skill trees, equipment system, health management
- Implementation: Data-driven approach with meaningful advancement

## Assets Required

### Post-Apocalyptic Assets
- Type: 3D Models and Textures
- Quantity: 150-250 decayed environments and survivor items
- Specifications: Consistent ruined aesthetic with atmospheric lighting

## Best Practices
- Design meaningful narrative choices with lasting consequences
- Create varied zombie behaviors to maintain tension
- Balance resource scarcity with player progression

## Common Pitfalls
- Making origin stories too linear or predictable
- Poorly balanced difficulty leading to frustration
