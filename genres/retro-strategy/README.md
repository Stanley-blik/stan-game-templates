# Retro Strategy Template

## Overview
A template for creating strategy games with retro aesthetics, classic gameplay mechanics, and nostalgic design elements.

## Core Requirements
1. Classic strategy gameplay with turn-based or real-time mechanics
2. Pixel art or retro visual style
3. Simple but deep strategic systems
4. Nostalgic audio and interface design

## Essential Systems

### Classic Strategy Mechanics
- Purpose: Implement timeless strategic gameplay elements
- Components: Unit movement, combat system, resource management
- Implementation: Grid-based or zone-based strategic interactions

### Retro Aesthetics
- Purpose: Maintain authentic retro visual and audio style
- Components: Pixel art renderer, chiptune audio, CRT effects
- Implementation: Shader-based effects with period-appropriate assets

## Assets Required

### Retro Assets
- Type: Pixel Art and Chiptune Audio
- Quantity: 50-100 pixel art assets and 10-20 musical pieces
- Specifications: Consistent retro style with limited color palettes

## Best Practices
- Study classic strategy games for design inspiration
- Create intuitive interfaces with clear strategic information
- Balance simplicity with strategic depth

## Common Pitfalls
- Overcomplicating simple retro mechanics
- Inconsistent visual style breaking immersion
